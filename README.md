# Data Wrangling Repository

## Purpose
This repository is designed to structure, clean, and prepare datasets for analysis and modeling. The focus is on building reproducible data pipelines that transform raw data into analysis-ready formats.

## Current Data Folders
- `data/app-logs/`: Contains application log data and related documentation for log transformation and analysis.
- `data/sales-data/`: Contains online sales data and related documentation for cleaning and forecasting.

## Workflow
1. Ingest raw dataset files from each folder.
2. Clean and preprocess the data.
3. Transform the data into consistent, structured formats.
4. Use the cleaned data for analysis, visualization, or forecasting.

## Notes
- Each dataset folder has its own README describing objectives and processing steps.
- The repository is intended as a central place for data wrangling work and pipeline documentation.
